# qb-switchfile

## Features

* switch file in different folder with the same name
* configable


## Requirements

vscode > 1.29.0

## Extension Settings

For example:

This extension contributes the following settings:

* `qb.fileSwitch.searchs`: folderA,folderB
* `myExtension.thing`: folderA, folderB
